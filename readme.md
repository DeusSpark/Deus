# Deus

My first big game

## Setup

Download SFML

### Debian

```
sudo apt install libsfml-dev
```

### Arch
```
sudo pacman -S SFML
```

## Clone

```
git clone https://github.com/DeusSpark/Deus.git
```

## Build

```
cd Deus &&
mkdir build &&
cd build &&
cmake .. &&
cmake --build . 
```

## Dependencies

  * [SFML](https://www.sfml-dev.org)
